# markdown-sticky-notes

## Features

- Draggable
- Resizable
- Save as you type
- Multiple editor themes, fonts

## Test locally

```js
yarn install
yarn build
```

Go to [chrome://extensions/](chrome://extensions/),click on "load unpacked", choose the `build` folder, then you can see the extension.
